## Money Honey

App capable to manage the financial status as expenses and income. In
additional, it sends reports by email and generate charts.

## Dependencies:

```
Go => 1.9.3
make => 4.2.1
```

## ENV

```
URL=http://exemple.local
USER=admin
PASS=123456
```

## Run the project:

To run locally, use `make test` to run the tests.
